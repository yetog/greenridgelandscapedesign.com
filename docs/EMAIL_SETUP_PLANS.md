## Option 1: Cloudflare Email Routing (Recommended)

**Best for:** Receiving email at custom domains, forwarding to Gmail
**Cost:** Free
**Complexity:** Easy

### Setup Steps
1. Add domain to Cloudflare (if not already)
2. Go to Email > Email Routing
3. Enable Email Routing
4. Add destination address (isayahy@gmail.com)
5. Create routing rules (e.g., *@yourdomain.com → Gmail)
6. Cloudflare auto-configures MX records

### Benefits
- No server maintenance
- High deliverability
- Can use Gmail "Send As" to reply from custom domain
- Free unlimited forwarding

### Limitations
- Can't host mailboxes (just forwarding)
- Need Gmail for actual inbox

## Option 2: Mailcow (Self-Hosted)

**Best for:** Full email hosting with webmail, multiple users
**Cost:** Free (server resources only)
**Complexity:** Medium

### Requirements
- Docker & Docker Compose
- Dedicated subdomain (e.g., mail.yourdomain.com)
- Ports 25, 465, 587, 993, 995 open
- Reverse DNS (PTR record) set correctly
- Clean IP (not blacklisted)

### Setup Steps
```bash
# 1. Clone Mailcow
cd /opt
git clone https://github.com/mailcow/mailcow-dockerized
cd mailcow-dockerized

# 2. Generate config
./generate_config.sh
# Enter mail hostname: mail.yourdomain.com

# 3. Start
docker-compose up -d

# 4. Access admin panel
# https://mail.yourdomain.com
# Default: admin / moohoo
```

### DNS Records Required
```
# MX Record
@       MX      10      mail.yourdomain.com

# A Record
mail    A       66.179.240.58

# SPF
@       TXT     "v=spf1 mx -all"

# DKIM (generated by Mailcow)
dkim._domainkey     TXT     "v=DKIM1; k=rsa; p=..."

# DMARC
_dmarc  TXT     "v=DMARC1; p=quarantine; rua=mailto:admin@yourdomain.com"
```

### Features
- Webmail (SOGo)
- Admin panel
- Spam filtering (Rspamd)
- Antivirus (ClamAV)
- Multiple domains & users
- CalDAV/CardDAV

### Challenges
- IP reputation (new IPs often flagged as spam)
- Ongoing maintenance
- Resource usage (~2GB RAM minimum)

---

## Recommendation

**Start with Cloudflare Email Routing** for receiving mail at custom domains. It's free, reliable, and requires no maintenance.

If you need full mailbox hosting later (multiple users, webmail), consider Mailcow on a dedicated VPS with a clean IP.
